## [腾讯发布视觉语言模型 Penguin-VL](https://huggingface.co/tencent/Penguin-VL-2B) `#2`
> 腾讯发布视觉语言模型 **Penguin-VL** 系列，包含 **2B**、**8B** 及独立编码器版本。该模型创新性地直接基于 `Qwen3` 纯文本模型构建视觉编码器，并引入时间冗余感知策略以优化长视频推理效率。

腾讯推出紧凑型视觉语言模型 **Penguin-VL**，旨在探索小规模 VLM 架构效率极限。该模型摒弃对比学习预训练，直接基于纯文本 `LLM` 构建视觉编码器，采用 `Qwen3` 主干。提供 **2B**、**8B** 及独立 Encoder 版本，可用时间为 **2025 年 3 月**。核心架构包含 `LLM` 初始化视觉编码器、`MLP` 投影器及双向注意力机制。针对长视频理解，引入 `TRA Token` 压缩策略。基准测试显示，**PenguinVL-2B** 在图表、文档及视频理解多项指标中领先同组竞品，但在部分 OCR 及数学推理测试中略低于 **Qwen3-VL 2B**。

## [Claude Code 新增 /Loop 命令支持会话内周期性任务](https://x.com/bcherny/status/2030193932404150413) `#3`
> **Claude Code** 推出 `/loop` 命令，支持在终端会话中设置最长 **三天** 的周期性自动任务，可用于监控代码构建、汇总消息等场景，但关闭终端后任务即失效。

**Claude Code** 官方正式发布全新的 `/loop` 命令，允许用户在会话期间按固定间隔自动执行指令，将 AI 转化为自动运行的值班助手。该功能支持通过自然语言或特定语法设置最长 **3 天**的周期性任务及一次性提醒，底层基于 `cron` 表达式实现，并引入抖动机制以优化 API 请求分布。单个会话最多可持有 **50** 个定时任务，任务触发以低优先级入队，不会打断正在进行的回复。

需注意，该功能仅绑定于当前终端会话，关闭终端后任务将失效且不会补跑，若需持久化任务需依赖 **GitHub Actions** 或桌面端定时功能。典型应用场景包括自动监控并修复 PR 构建错误、通过 **Slack MCP** 汇总消息等。

## [阿里巴巴武汉大学联合推出 AgeMem 记忆框架](https://arxiv.org/abs/2601.01885) `#6`
> 阿里巴巴与武汉大学研究团队近期联合推出 **AgeMem** 框架，将记忆管理直接嵌入大语言模型智能体的决策流程。该系统通过强化学习端到端训练，使模型能自主决定信息的存取与更新，无需依赖预设规则。

阿里巴巴与武汉大学研究团队近期合作推出 **Agentic Memory (AgeMem)** 统一框架，将长期和短期记忆管理直接集成至大型语言模型 `Agent` 策略中。该框架将记忆操作转化为基于工具的动作，支持添加、删除、更新等指令，使模型能自主决定信息存储与丢弃时机，解决有限上下文窗口限制。系统采用强化学习进行端到端训练，提出三阶段渐进式策略及步进式 `GRPO`，解决奖励稀疏问题。在五个长跨度基准测试中，**AgeMem** 在多个 `LLM` 主干网络对比中均优于当前强记忆增强基准模型。指标显示，该框架实现了任务性能实质提升，获得更高质量长期记忆及更高效上下文使用表现。

## [智谱发布 GLM-5-Turbo，专为 OpenClaw 等 Agent 驱动环境优化](https://docs.bigmodel.cn/cn/guide/models/text/glm-5-turbo) `#1`
> 智谱发布了专为 **OpenClaw** 等 Agent 驱动环境优化的高速模型 `GLM-5-Turbo`。
>
> 该模型 API 定价比 `GLM-5` 略高。在 `GLM Coding Plan` 中，Max 套餐用户已可直接使用，Pro 用户将在本月底前开放，Lite 用户将在下月可用。

智谱发布 **GLM-5-Turbo**，该模型是基于 **GLM-5** 专为 `OpenClaw` 等 Agent 驱动环境进行了优化的高速版本，由内部代号“pony-alpha-2”的项目升级而来。官方称，此实验性模型暂不开源，成果将整合至下一代开源版本。

在 **GLM Coding Plan** 中，`Max` 套餐现已支持，`Pro` 套餐本月月底前开放，`Lite` 套餐预计 **4** 月接入。

计费上，高峰期（**14:00-18:00 UTC+8**）按 **3** 倍额度消耗，非高峰期按 **2** 倍。截至 **4** 月底，非高峰期可享 **1** 倍系数限时福利。

## 智谱开启 GLM Coding Plan 限时服务窗口保障老用户权益 `#2`
> 智谱针对 **GLM Coding Plan** 开启持续到本月底的限时服务窗口，专门处理老用户权益问题。
>
> 在此期间，符合条件的老用户可续订已停售的老版套餐，误升级的用户支持回退且费用“多退少不补”。
>
> 针对部分用户被系统漏算优惠价值的情况，官方将按漏算金额的 **1.5倍** 发放赠金补偿。

智谱宣布针对 **GLM Coding Plan** 开启限时服务窗口，该窗口昨日开启并将持续至本月底。此次方案旨在保障老用户权益，涵盖续订、回退及补偿三项内容。

对于停售前订阅且到期日晚于规定日期的用户，可续订同等级同周期老版套餐，原剩余价值按天折算抵扣。

针对停售初期且新说明上线前误升级用户，官方支持回退原套餐，费用差额遵循多退少不补原则。

此外，因系统原因导致优惠剩余价值漏算的用户，官方将按漏算金额的 **1.5 倍** 发放赠金补偿。

智谱表示将持续优化服务细节。

>

每类月卡限购 **5 张**，开发者可使用 API Key 鉴权调用。

企业大规模需求可联系销售定制。
